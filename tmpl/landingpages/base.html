About the site

Hi, my name is Dwight and Knownly is my little creation.

I first had the idea to host websites straight from Dropbox a while back in January 2014. I was reading through the Dropbox API docs for a project and the light bulb went on...

It turned out to be a great way to quickly put landing pages online for new ideas. I've done away with all the Wordpress blogs I'd been roped in to managing over the years. After sharing Knownly with a few friends I learned it's also useful to other creatives, designers and developers (plus people wanting to learn these skills). The best part is we don't need to worry about web hosting and control panels anymore.

So it's time to launch properly - to make it known. As I said, I hope it's useful to you and I love to hear feedback :)
